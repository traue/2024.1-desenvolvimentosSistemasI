Exercício

Fazer um programa para ler os dados de um produto em um sistema de estoque. 
Cada produto possui nome, uma descrição, preço e quantidade disponível. 

O programa deverá:
 - Mostrar os dados do produto (nome, descrição, preço, quantidade em estoque, valor total no estoque)
 - Realizar uma entrada no estoque 
 - Realizar uma saída no estoque 

Para resolver este problema, você deve criar uma CLASSE conforme diagrama proposto.
